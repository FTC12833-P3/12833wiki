Scrcpy (screen copy)


To set up screen copy you need to go https://github.com/Genymobile/scrcpy#readme

Download the latest version

Extract the .exe

Now you can run scrcpy

To see the robot's camera view:

Connect to the robot's wifi

Open REV Hardware client and connect

Open Scrcpy.exe

This is ideally used for streaming the actual camera feed, so you can see the apriltag detection in real time.
