Telemetry


To set up multiple telemetry your directions change for how your code is set up. If you have an abstract opMode setup you will need to add this line to your abstract opMode:

public MultipleTelemetry multipleTelemetry = new MultipleTelemetry(this.telemetry, FtcDashboard.getInstance().getTelemetry());

This line creates a new multiple telemetry object using the original telemetry and the ftc Dashboard telemetry as parameters, the 2 places it will output

If you are using the LinearOpMode, you will need to add the to every opMode where you use multipleTelemetry;

NOTE: Any telemetry you want to graph should appear at least once in init, so you can select it before starting the code
