Dashboard


How to set up FTC Dashboard:

(I assume you have a working ftc robot controller version)

1. Open build.dependencies.gradle. In the repositories section add maven { url = 'https://maven.brott.dev/' }

2. In the dependencies section add implementation 'com.acmerobotics.dashboard:dashboard:X.X.X' // Fill in X.X.X with the latest dashboard version

3. Now gradle sync

Done!

To access the dashboard:

Connect to the robot wifi

Open web browser and go to this address http://192.168.43.1:8080/dash
