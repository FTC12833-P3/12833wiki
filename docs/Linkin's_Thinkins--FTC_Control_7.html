FTC Control


FTC control is a new dashboard that is trying to replace ftc Dashboard. Currently it has issues with most everything.

The hardest part is to actually download it to your project, then to add limelight is even more to do.

The biggest reason for using this currently is for limelight streaming.

FTC dashboard has also added this feature, but currently have no documentation on it.

Ftc Control is able to run opModes, draw your robot on a field map, and camera stream.

Currently I'd say that it is not worth the effort, but is something to keep an eye on. I figure that it will be much less stable than dashboard at least for a couple years and has a much smaller user base.

Here is the website: https://ftcontrol.bylazar.com/docs/overview/
